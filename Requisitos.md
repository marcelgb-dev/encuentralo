# Localizador de oportunidades comerciales

Herramienta web que permita realizar búsquedas en fuentes externas como Google Maps, visualizar los resultados de forma estructurada y guardarlos para consultarlos más adelante.

- [x] El usuario podrá introducir un término de búsqueda y seleccionar la fuente de datos a consultar (Google Maps, Apple Maps o similares).
- [x] Mediante la ubicación, se geolocalizará al usuario y se buscará alrededor suya el termino que el ha introducido.
- [x] Los resultados se mostrarán de forma estructurada: nombre, dirección, categoría, valoración u otros datos relevantes.
- [ ] Los resultados se guardarán en una base de datos para no consultar repetitivamente sobre la misma consulta.
- [ ] El usuario podrá filtrar u ordenar los resultados por diferentes criterios (nombre, valoración, categoría…).
- [ ] También se podrán guardar resultados individuales en una lista de favoritos propia.
- [ ] Las búsquedas guardadas mostrarán la fecha en que se realizaron y el término usado.
- [ ] Se mostrará un historial con las últimas búsquedas realizadas, aunque no se hayan guardado. 
- [ ] El usuario podrá etiquetar o clasificar los resultados guardados por categorías propias. 
- [ ] Se podrá exportar el listado de resultados guardados en formato CSV o similar. 
- [ ] La aplicación requerirá registro e inicio de sesión; cada usuario tendrá su propio espacio privado.
